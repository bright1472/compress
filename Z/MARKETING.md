# Titan 视频压缩 — 产品介绍

## 一句话介绍

浏览器内直接压缩视频，无需上传到服务器，保护隐私。

## 核心优势

### 隐私安全
视频文件**永远不离开你的电脑**。整个压缩过程在浏览器本地完成，
不上传到任何服务器。没有文件大小限制，没有等待队列，没有数据泄露风险。

### 三引擎智能路由

| 模式 | 适用场景 | 速度 | 需要安装 |
|------|---------|------|---------|
| **WebCodecs**（默认） | 1080p 及以下视频 | GPU 硬件加速，约 1× 实时 | 否 |
| **Native Host**（极速） | 批量压缩 / 2K 以上 | GPU 硬件加速，10-50× 实时 | 是（可选） |
| **FFmpeg WASM**（兜底） | 老旧浏览器 | CPU 编码，稳定可用 | 否 |

系统自动检测设备能力，选择最优引擎，用户无需手动选择。

### 支持格式

**输入**：MP4、MOV、MKV、AVI、WebM、FLV、WMV、3GP、OGV、M4V、TS、MTS

**输出**：MP4（H.264 / H.265 / AV1 可选）

### 核心功能

- **智能码率计算**：自动分析原视频码率，避免"越压越大"
- **质量预设**：CRF 滑块（18-40），从"肉眼无损"到"极限压缩"
- **速度预设**：ultrafast / fast / medium / slow 四档可选
- **批量队列**：一次添加多个文件，自动排队处理
- **压缩前后对比**：内置滑块对比，直观查看画质变化
- **深色/亮色主题**：自动跟随系统，手动切换持久化

## 常见问题

### 压缩速度有多快？

**1080p 视频**：利用浏览器 GPU 硬件加速，速度明显快于 CPU 编码（实测 640×480 达 279% 实时）。
具体速度取决于 GPU 型号和视频分辨率，通常显著快于纯软件编码。

**2K / 4K 视频**：浏览器内置 GPU 编码器不支持高分辨率，自动降级为 CPU 软件编码。
速度约为 2-3 MB/s（取决于 CPU 性能）。

如果需要 2K/4K 的 GPU 硬件加速，可安装可选的 **Native Host 极速模式**，
速度提升至 10-50× 实时（同一段 2K 视频从 15 分钟缩短到 30 秒）。

### 支持多大的文件？

Web 模式下，文件大小**无硬性限制**。使用 OPFS（Origin Private File System）
流式写入，不会因文件过大导致内存溢出。我们已测试过 1GB+ 文件的压缩。

### 需要联网吗？

首次加载需要联网（加载 WASM 和 mediabunny 库），之后压缩过程**完全离线**，
不依赖任何远程服务器。

### 和在线压缩工具有什么区别？

| | Titan | 在线压缩工具 |
|--|-------|-------------|
| 隐私 | 文件不离开电脑 | 需上传到对方服务器 |
| 大小限制 | 无限制 | 通常 500MB-2GB |
| 速度 | 取决于本机硬件 | 取决于上传带宽 + 排队 |
| 费用 | 免费 | 通常有付费墙 |

## 快速开始

1. 打开 Titan 网页
2. 拖拽或点击上传视频文件
3. 选择编码格式和质量（或使用默认设置）
4. 点击开始压缩，等待完成
5. 下载压缩后的文件，或使用对比滑块查看效果

### 极速模式（可选）

如需 2K/4K GPU 硬件加速和批量压缩能力：

1. 安装 Native Host 程序（见 [NATIVE-GUIDE.md](NATIVE-GUIDE.md)）
2. 加载 Chrome 扩展（`native-bridge/` 目录）
3. 打开 Titan 网页，自动识别并显示 极速模式 入口

## 浏览器要求

- **推荐**：Chrome 120+ / Edge 120+（完整支持 WebCodecs + GPU 加速）
- **兼容**：Firefox（降级到 FFmpeg WASM，无 GPU 加速）
- **必需响应头**：服务器需返回 `COOP: same-origin` + `COEP: require-corp`
